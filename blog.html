What is Document Object Model(DOM)?

The DOM (Document Object Model) is a way for web browsers to represent and interact with the structure of a webpage. It turns the HTML code of a page into a tree-like structure, where each part of the page (like elements, text, and attributes) is represented as an object. With the DOM, JavaScript can read and change the content, style, and structure of a webpage while itâ€™s being viewed, allowing for dynamic updates and interactivity.

What is event Delegation in the context of the DOM and why it is useful?

Event delegation is a technique in the DOM where you add a single event listener to a parent element, instead of adding multiple event listeners to individual child elements. The event listener listens for events on its children through event bubbling (where events propagate up the DOM tree).


Why it is useful:

  1. 1.Efficiency: Instead of adding event listeners to many elements, you add one listener to a common parent.
  2. 2.Dynamic elements: It handles events for elements added later (after the page loads), without needing new event listeners.
How do you manipulate an element's attributes and styles using the DOM?

We can Manipulate HTML element by using some method and we can also set some attributes and style as we like.Javascript gives us the full potential to do whatever we want.


  • 1. Attribute Manipulation: getAttribute(attributeName): Retrieves the value of a specified attribute. setAttribute(attributeName, value): Sets or updates the value of an attribute. removeAttribute(attributeName): Removes an attribute from an element.
  • 2. Style Manipulation: element.style.property = value: Directly sets a specific CSS style property, such as element.style.color = 'red';.
  • 3. Class Manipulation: classList.add(className): Adds a new CSS class to the element. classList.remove(className): Removes a CSS class from the element. classList.toggle(className): Toggles the presence of a CSS class (adds if missing, removes if present).
